# Minggu 5 - Analisis Dinamis

## Materi
Pada pertemuan kelima saya mempelajari analisis dinamis menggunakan debugger untuk mengamati jalannya program secara langsung ketika dieksekusi.

## Tujuan Pembelajaran
- Memahami konsep analisis dinamis.
- Mengenal fungsi debugger.
- Mengamati alur eksekusi program.

## Tools yang Digunakan
- x64dbg
- GDB

## Hasil Pembelajaran
Saya memahami bagaimana debugger digunakan untuk melihat isi register, memori, serta proses eksekusi instruksi pada sebuah program.

## Kesimpulan
Analisis dinamis memberikan informasi yang tidak dapat diperoleh hanya melalui analisis statis sehingga keduanya saling melengkapi.
