# Minggu 2 - Arsitektur Komputer dan Memori

## Materi
Pada pertemuan kedua saya mempelajari dasar arsitektur komputer yang berhubungan dengan Reverse Engineering. Materi meliputi CPU, register, memori, stack, serta perbedaan pengelolaan memori pada bahasa pemrograman C dan Java.

## Tujuan Pembelajaran
- Memahami fungsi CPU dalam menjalankan instruksi.
- Mengetahui peran register dan stack.
- Memahami konsep pengelolaan memori pada C dan Java.

## Tools yang Digunakan
- WASKITA LMS
- Ghidra

## Hasil Pembelajaran
Saya memahami bahwa CPU menjalankan instruksi dengan memanfaatkan register dan memori. Saya juga mengetahui bahwa bahasa C menggunakan manajemen memori secara manual sehingga lebih berisiko mengalami memory corruption, sedangkan Java menggunakan Garbage Collection yang mengelola memori secara otomatis.

## Kesimpulan
Materi arsitektur komputer menjadi dasar penting dalam Reverse Engineering karena membantu memahami bagaimana sebuah program dijalankan dan bagaimana data disimpan di dalam memori.
