# Minggu 6 - Identifikasi Algoritma dari Binary

## Materi
Pada pertemuan keenam saya mempelajari cara mengidentifikasi algoritma yang digunakan dalam sebuah program melalui analisis kode binary.

## Tujuan Pembelajaran
- Memahami pola algoritma pada binary.
- Mengidentifikasi logika program.
- Melakukan rekonstruksi algoritma sederhana.

## Tools yang Digunakan
- Ghidra
- x64dbg

## Hasil Pembelajaran
Saya mempelajari bagaimana mengenali struktur perulangan, percabangan, dan fungsi sehingga dapat memahami logika program meskipun tidak memiliki source code.

## Kesimpulan
Identifikasi algoritma membantu memahami cara kerja sebuah program dan menjadi langkah penting dalam proses Reverse Engineering.
