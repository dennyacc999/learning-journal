# Minggu 3 - Binary dan Executable

## Materi
Pada pertemuan ketiga saya mempelajari konsep binary dan executable sebagai dasar dalam Reverse Engineering. Materi yang dipelajari meliputi proses pembentukan executable, format file PE dan ELF, struktur internal executable, serta konsep endianness.

## Tujuan Pembelajaran
- Memahami pengertian binary dan executable.
- Mengetahui proses pembentukan file executable.
- Memahami struktur file PE dan ELF.
- Mengetahui fungsi setiap section pada file executable.

## Tools yang Digunakan
- WASKITA LMS
- Ghidra
- HxD

## Hasil Pembelajaran
Saya memahami bahwa executable merupakan hasil kompilasi source code yang dapat dijalankan oleh sistem operasi. Saya juga mempelajari struktur file Portable Executable (PE) pada Windows yang terdiri dari Header, .text, .data, dan .rsrc. Selain itu, saya memahami pentingnya konsep endianness dalam membaca data binary.

## Kesimpulan
Melalui materi ini saya lebih memahami bagaimana sebuah file executable disusun dan bagaimana informasi di dalamnya dapat dianalisis menggunakan tools Reverse Engineering.
