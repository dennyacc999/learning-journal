# Minggu 4 - Analisis Statis Binary

## Materi
Pada pertemuan keempat saya mempelajari teknik analisis statis terhadap file binary. Analisis dilakukan tanpa menjalankan program menggunakan beberapa tools seperti Strings, objdump, dan Ghidra.

## Tujuan Pembelajaran
- Memahami konsep analisis statis.
- Mengenal fungsi tools analisis statis.
- Mengidentifikasi informasi penting dari file binary.

## Tools yang Digunakan
- Strings
- Ghidra
- objdump

## Hasil Pembelajaran
Saya mempelajari cara melihat informasi dasar sebuah file executable, daftar fungsi, library yang digunakan, serta string penting yang terdapat di dalam program tanpa menjalankannya.

## Kesimpulan
Analisis statis membantu memahami struktur dan perilaku awal suatu program sebelum dilakukan analisis lebih lanjut.
