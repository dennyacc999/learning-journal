# Learning Journal Reverse Engineering

## Deskripsi

Repository ini berisi dokumentasi proses pembelajaran mata kuliah Reverse Engineering selama satu semester.

## Tujuan

* Mendokumentasikan materi yang dipelajari setiap minggu.
* Mencatat penggunaan tools Reverse Engineering.
* Menyimpan hasil latihan dan analisis sederhana.
* Membangun portofolio pembelajaran menggunakan GitHub.

## Isi Repository

Repository ini berisi jurnal mingguan yang disusun berdasarkan materi perkuliahan Reverse Engineering. Setiap jurnal memuat ringkasan materi, tools yang digunakan, hasil pembelajaran, dan kesimpulan dari setiap pertemuan.

Repository ini dibuat sebagai bagian dari tugas portofolio mata kuliah Reverse Engineering.
